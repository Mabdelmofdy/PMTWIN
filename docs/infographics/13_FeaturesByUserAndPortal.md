## Portal-Specific Workflows

### Public Portal Workflow

```mermaid
flowchart TD
    Start([Visitor Arrives]) --> Browse[Browse Public Portal]
    
    Browse --> Options{What to Do?}
    
    Options -->|Discover Projects| Discovery[Project Discovery<br/>Limited Visibility]
    Options -->|Learn About Models| Wizard[PMTwin Wizard<br/>Interactive Guide]
    Options -->|Access Resources| Knowledge[Knowledge Hub<br/>Educational Content]
    Options -->|Join Platform| Register[Registration]
    
    Discovery --> ViewLimited[View Limited Project Info]
    ViewLimited --> Register
    
    Wizard --> LearnModels[Learn About 5 Models]
    LearnModels --> Register
    
    Knowledge --> ReadContent[Read Articles/Guides]
    ReadContent --> Register
    
    Register --> SelectRole{Select Role}
    SelectRole -->|Individual| RegIndividual[Register as Individual]
    SelectRole -->|Entity| RegEntity[Register as Entity]
    
    RegIndividual --> UploadCreds[Upload Credentials]
    RegEntity --> UploadCreds
    
    UploadCreds --> Submit[Submit Registration]
    Submit --> Pending[Status: Pending Approval]
    Pending --> Wait[Wait for Admin Approval]
```

**Available Features:**
- Project Discovery (limited visibility)
- PMTwin Wizard (interactive model selection guide)
- Knowledge Hub (educational resources)
- Registration (Individual or Entity)
- View public project listings (basic info only)

**Restrictions:**
- Cannot view full project details
- Cannot submit proposals
- Cannot access user features
- Cannot create projects
